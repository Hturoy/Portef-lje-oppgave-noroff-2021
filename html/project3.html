The museum

The assignment

In this assignment we were tasked to make a website for a fictional museum with the target audience being primary and middle school children and families with young children.

The process

I started the assignment by creating a deisgn in Adobe XD, when i was happy with it i started building the website, in this particular assignment we were only using HTML and CSS.

Changes

If i were to update this website i would redo the design, change the colors and font, also i would add more design elements to make the website more interesting to the target audience.

I would also implement some javascript to make the website more responsive and mobilefriendly, like adding a Hamburger menu when visiting from a phone
