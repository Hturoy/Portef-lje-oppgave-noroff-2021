The blog

The assignment

In this assignment we were tasked to make a blog with a theme of our choosing, as well as creating blogposts we were also tasked with creating a few different javascript functions like a carousel and a modal for the images.

The process

I started this assignment by planning out an order to do the diffrent parts of the project. I decided to start with making the javascript functions, since these would most likely take the most time to complete. after the javascript i started designing in XD and building the rest of the site.

Changes

The first change i would make to the website would be to create a bigger carousel while visiting the blog. I would also add more design elements to the website, like a background to the front page, and i would change the layout of the blogposts page to have 2-3 collumns instead of 1.
