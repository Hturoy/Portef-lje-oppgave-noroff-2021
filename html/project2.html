Rainy Days

The assignment

In this assignment we were tasked to choose between three diffrent websites to make, i chose to make a website for Rainy days. in this assignment we had to create a website where you can buy the Rainy days rain coats.

The process

Like the museum assignment i started by sketching and prototyping in Adobe XD, when i was happy with the design i started building using HTML and CSS

Changes

The changes i would make to this website are very similar to the first project, i woul redo the design and layout of the website, also i would use javascript to add a carousel on the front page with "Popular" items as well as a burger menu and clickable pictures so that you can increase the size of the photos for a closer look.
